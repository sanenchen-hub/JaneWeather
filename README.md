## 使用说明
### 直接使用
APK文件：[下载链接](https://github.com/sanenchen-hub/JaneWeather/releases/tag/0.1.1)

### 自行编译
1. 申请一个和风天气API
2. 申请一个高德地图API

申请完了后，把和风天气API写到apiKey/ContentUtils
![](https://blogoss.lyqmc.cn/typecho/uploads/2020/08/15982007578417.jpg)

再放入高德地图Key

之后就运行吧
